matchRouter

matchRouter

根据当前路径匹配router配置。

产生结果：req.param, req.router。 若router设置请求方法与http method不匹配，响应浏览器405 error。 req.param会被分别合并到req.query, req.body。

查看结果：

const stage = coc(app);
stage.after('matchRouter', (req, res, next) => {
  console.log(req.router, req.param);
  next();
});
